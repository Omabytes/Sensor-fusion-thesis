\chapter{Simulated Average Error Raw Values}
\label{app:rawvalues}

\initial{T}ables \ref{tab:rawvalues} to \ref{tab:rawvaluesnosensordouble} give the simulation output values for average error with and without inclusion of the sensory apparatus in the filter. Average error is described as the mean distance from the true simulated value each filter time step over a large number of simulation runs. Table \ref{tab:rawvaluesnosensordouble} shows the affect that doubling simulation run time has on state estimates when no sensory apparatus input is available.
